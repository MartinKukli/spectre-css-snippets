# Spectre.css Snippets

VSCode snippets extension for [Spectre.css](https://picturepan2.github.io/spectre/)

## Features

Get list of Spectre.css classes by typing 'spectre:'.

## Currently supported

- Elements
- Layout

## Planned features

- Snippets for Components, Utilities and Experimental
- Premade html blocks
